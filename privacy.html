Privacy Policy

Last updated: February 2026

Short version

I don't collect any personal data. No cookies, no tracking scripts, no analytics. This site is just static HTML files hosted on GitHub Pages.

Data collection

This website does not collect, store, or process any personal information. Specifically:

  • No cookies: We don't use cookies or local storage
  • No tracking: No Google Analytics, no Facebook pixels, no tracking scripts
  • No user accounts: There's no login or registration system
  • No email collection: No newsletter signup or contact forms

Third-party services

This site uses the following third-party services:

  • GitHub Pages: Hosting service. GitHub may log access requests as part of their normal operations.
  • CDN resources: Tailwind CSS and Prism.js are loaded from public CDNs (cloudflare.com, cdnjs.com).

These services have their own privacy policies. I encourage you to review them if you're concerned about data handling.

External links

This site contains links to external websites like GitHub, various documentation sites, and other resources. Once you click on these links, you're leaving this site and subject to the privacy policies of those external sites.

I'm not responsible for the privacy practices of any third-party websites.

Comments and feedback

If you leave a comment on GitHub (where the source code is hosted), any information you provide (your GitHub username, the content of your comment) will be public and visible to anyone.

Don't include personal information like email addresses, phone numbers, or addresses in GitHub comments.

Children's privacy

This site is not directed to children under 13. Since I don't collect personal information anyway, this isn't really applicable, but I wanted to mention it for completeness.

Changes to this policy

I might update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.

Given the nature of this site (static blog with no data collection), don't expect major changes to this policy.

Contact

If you have questions about this privacy policy, feel free to reach out through GitHub.
